# 读书评价摘抄云端部署说明

目标：部署后得到一个公网网址，别人打开这个网址就能添加书籍、评价、摘抄，并共享同一份数据。

## 推荐方式：Render

1. 注册或登录 Render。
2. 把本文件夹上传到 GitHub 仓库。
3. 在 Render 选择 New -> Blueprint。
4. 选择这个 GitHub 仓库。
5. Render 会读取 `render.yaml` 自动创建网站。
6. 部署完成后，会得到一个类似下面的网址：

   `https://reading-review-notebook.onrender.com`

7. 把这个网址发给别人即可。

## 数据保存

应用会把数据保存到服务器上的 `reading-data.json`。

在 Render 上，`render.yaml` 已经配置了 1GB 磁盘：

```yaml
DATA_DIR=/var/data
```

所以重启后数据仍会保留。

## 本地运行

```bash
npm start
```

然后打开：

```text
http://127.0.0.1:8765/index.html
```

## 注意

- `127.0.0.1` 只适合本机。
- 公网部署后，用 Render 给你的 `https://...` 地址分享给别人。
- 现在是共享数据模式，所有打开同一个网址的人看到的是同一套书架和摘抄。
